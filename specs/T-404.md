# T-404: Copy-to-Clipboard für vollständiges TXT-Transkript implementieren

## Kontext
Status: review
Priorität: medium
Parent Task: T-400
Abhängigkeiten: T-400, T-304

## Goal
Vollständiges Transkript als TXT in Clipboard kopieren können.

## Done When
- [x] Copy-Action erzeugt vollständigen Text aller finalen Segmente.
- [x] Format enthält Metadaten-Header (Datum/Startzeit/Dauer) sowie Sprecher/Uhrzeit/Text.
- [x] Erfolg/Fehler beim Kopieren wird dem Nutzer angezeigt.

## Approach
- Export-String im Main oder Renderer zentral formatieren.
- Clipboard-API robust mit Fallback verwenden.
- Deutsches Zeitformat einhalten.

## Log
- 2026-06-03: Spec detailliert ausgefüllt.
- 2026-06-18: Exportformat erweitert: vor der Segmentliste wird ein Metadatenblock (`datum`, `startzeit`, `dauer`) generiert; Segmente werden als Liste mit deutschem Zeitformat in die Zwischenablage kopiert.
