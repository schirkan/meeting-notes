# T-404: Copy-to-Clipboard für vollständiges TXT-Transkript implementieren

## Kontext
Status: open
Priorität: medium
Parent Task: T-400
Abhängigkeiten: T-400, T-304

## Goal
Vollständiges Transkript als TXT in Clipboard kopieren können.

## Done When
- [ ] Copy-Action erzeugt vollständigen Text aller finalen Segmente.
- [ ] Format enthält Quelle/Sprecher/Uhrzeit/Text.
- [ ] Erfolg/Fehler beim Kopieren wird dem Nutzer angezeigt.

## Approach
- Export-String im Main oder Renderer zentral formatieren.
- Clipboard-API robust mit Fallback verwenden.
- Deutsches Zeitformat einhalten.
